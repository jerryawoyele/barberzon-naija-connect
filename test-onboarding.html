Onboarding Test

Open browser console (F12) and run the commands below to simulate onboarding completion:

Test Commands:

1. Check current user:

console.log('Current user:', JSON.parse(localStorage.getItem('user') || '{}'));

2. Complete onboarding as Customer:

const currentUser = JSON.parse(localStorage.getItem('user') || '{}'); const updatedUser = { ...currentUser, role: 'CUSTOMER', completedOnboarding: true, onboardingData: { userType: 'customer', locationLat: 6.5244, locationLng: 3.3792 } }; localStorage.setItem('user', JSON.stringify(updatedUser)); console.log('Updated user:', updatedUser); window.location.reload();

3. Complete onboarding as Barber:

const currentUser = JSON.parse(localStorage.getItem('user') || '{}'); const updatedUser = { ...currentUser, role: 'BARBER', completedOnboarding: true, onboardingData: { userType: 'barber', specialties: ['Fade', 'Beard Trim'], hourlyRate: '5000' } }; localStorage.setItem('user', JSON.stringify(updatedUser)); console.log('Updated user:', updatedUser); window.location.reload();

4. Reset onboarding (to test again):

const currentUser = JSON.parse(localStorage.getItem('user') || '{}'); const resetUser = { ...currentUser, completedOnboarding: false }; localStorage.setItem('user', JSON.stringify(resetUser)); console.log('Reset user:', resetUser); window.location.reload();
